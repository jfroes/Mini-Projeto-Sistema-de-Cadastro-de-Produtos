## Objetivo
Construir um sistema CRUD completo em Java POO que permita **cadastrar, listar, atualizar e excluir produtos**.

___

## Especificação Técnica
- Linguagem: Java 17+
- Paradigma: Programação Orientada a Objetos
- Simulado em lista (ArrayList)
- Foco: organização em camadas, encapsulamento, métodos, e tratamento básico de erros.
